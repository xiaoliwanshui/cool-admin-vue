## 📖 项目概述

Cool Admin Vue 是一个现代化的后台权限管理系统，基于 **Vue3 + TypeScript + Vite** 构建，开源免费，支持模块化、插件化架构，适用于快速构建企业级后台管理系统。

### 🎯 项目特点

- **🤖 AI 编码**：通过大模型生成 API 接口与前端页面，实现一键生成功能
- **🔄 流程编排**：拖拽式流程设计，支持智能客服等复杂业务流程
- **🧩 模块化设计**：代码结构清晰，组织合理，便于维护和扩展
- **🔌 插件化架构**：支持支付、短信、邮件等功能插件的灵活扩展
- **🌍 国际化支持**：内置中文、繁体中文、英文三种语言
- **⚡ 极速开发**：快速 CRUD 生成，大幅提升开发效率

```
cool-admin-vue/

│   │   ├── dict/         # 字典管理模块
│   │   ├── task/         # 任务管理模块
│   │   ├── video/        # 视频管理模块

│   ├── plugins/          # 插件目录
│   │   ├── crud/         # CRUD 快速开发插件
│   │   ├── i18n/         # 国际化插件
│   │   ├── theme/        # 主题插件
│   │   ├── upload/       # 文件上传插件
│   │   └── ...           # 其他功能插件
│   ├── cool/             # 核心框架
│   │   ├── bootstrap/    # 启动引导
│   │   ├── hooks/        # 钩子函数
│   │   ├── router/       # 路由管理
│   │   ├── service/      # 服务层
│   │   └── utils/        # 工具函数
│   └── config/           # 配置文件
├── packages/             # 自研包源码
│   ├── crud/             # CRUD 组件包
│   └── vite-plugin/      # Vite 插件包
└── public/               # 静态资源
```

## 🚀 快速开始

### 环境要求

- **Node.js**: 推荐使用 LTS 版本（≥ 16.x）
- **pnpm**: 推荐使用 pnpm 作为包管理器
- **Git**: 版本控制工具

### 安装依赖

```bash
# 安装项目依赖
pnpm install
```

### 启动开发服务器

```bash
# 启动开发模式
pnpm dev
```

访问地址：[http://localhost:9000](http://localhost:9000)

### 构建生产版本

```bash
# 构建生产环境
pnpm build

# 构建静态模式
pnpm build-static

# 构建演示模式
pnpm build-demo
```

### 代码检查与格式化

```bash
# ESLint 检查并自动修复
pnpm lint

# Prettier 格式化代码
pnpm format

# TypeScript 类型检查
pnpm type-check
```

## 📦 核心模块介绍

### 1. 基础模块 (base)

**功能**：提供系统基础功能，包括用户认证、权限控制、菜单管理等

**主要特性**：
- 用户登录/登出
- 权限验证中间件
- 动态菜单生成
- 个人信息管理
- 错误页面处理

### 2. 用户管理模块 (user)

**功能**：完整的用户管理系统

**主要特性**：
- 用户列表管理
- 角色权限分配
- 部门组织架构
- 用户信息维护

### 3. 字典管理模块 (dict)

**功能**：系统字典数据管理

**主要特性**：
- 数据字典维护
- 分类管理
- 缓存机制
- 多级字典支持

### 4. 任务管理模块 (task)

**功能**：后台任务调度管理

**主要特性**：
- 定时任务配置
- 任务执行监控
- 执行日志查看
- 任务状态管理

### 5. 视频管理模块 (video)

**功能**：视频内容管理系统

**主要特性**：
- 视频上传管理
- 播放器集成
- 视频信息维护
- 分类标签管理

## 🔌 插件系统

### CRUD 插件
**描述**：快速 CRUD 开发插件，大幅提升开发效率

**特性**：
- 表格组件自动生成
- 表单验证规则配置
- 搜索条件构建
- 分页处理
- 批量操作支持

### 国际化插件 (i18n)
**描述**：多语言支持插件

**配置**：
- 支持语言：中文(zh-cn)、繁体中文(zh-tw)、英文(en)
- 配置文件：`src/plugins/i18n/config.ts`
- 翻译文件：各模块的 `locales` 目录

**使用方式**：
```typescript
// 组件中使用
import { useI18n } from 'vue-i18n';
const { t } = useI18n();

// 模板中使用
{{ t('key') }} 或 {{ $t('key') }}
```

### 主题插件 (theme)
**描述**：主题切换和样式定制插件

**特性**：
- 多主题支持
- 动态主题切换
- 自定义主题配置
- 暗黑模式支持

### 文件上传插件 (upload)
**描述**：文件上传处理插件

**特性**：
- 多种上传方式
- 文件类型限制
- 上传进度显示
- 图片预览功能

### 图表插件 (echarts)
**描述**：数据可视化图表插件

**特性**：
- ECharts 集成
- 响应式图表
- 多种图表类型
- 实时数据更新

## 🛠️ 开发规范

### TypeScript 配置

项目使用严格的 TypeScript 配置：
- `noImplicitAny: false` - 允许隐式 any 类型
- `experimentalDecorators: true` - 启用装饰器支持
- 路径别名配置：
  - `/@/*` → `./src/*`
  - `/$/*` → `./src/modules/*`
  - `/#/*` → `./src/plugins/*`
  - `/~/*` → `./packages/*`

### 代码规范

- **ESLint + Prettier** 自动格式化
- **Vue3 Composition API** 优先使用
- **模块化开发** 按功能划分模块
- **组件化设计** 可复用组件抽离

### 国际化规范

- 所有 UI 文本必须使用 i18n 函数
- 翻译键使用简短、描述性的中文命名
- 新功能开发时同步添加多语言支持

## 🔧 配置说明

### 环境配置

项目支持多环境配置：

- **开发环境** (`dev.ts`)：本地开发配置
- **生产环境** (`prod.ts`)：生产部署配置
- **代理配置** (`proxy.ts`)：API 代理设置

### Vite 配置

主要配置项：
- **端口**：9000
- **代理**：支持 API 代理配置
- **别名**：路径别名简化导入
- **插件**：集成 Vue、压缩、i18n 等插件
- **构建优化**：代码分割、资源优化

### 应用配置

```typescript
{
  app: {
    name: "Cool Admin",           // 应用名称
    menu: {
      isGroup: false,             // 菜单是否分组显示
      list: []                    // 自定义菜单列表
    },
    router: {
      mode: 'history',            // 路由模式
      transition: 'slide'         // 转场动画
    }
  },
  i18n: {
    locale: 'zh-cn',              // 默认语言
    languages: [...]              // 支持的语言列表
  }
}
```

## 🚀 部署指南

### Docker 部署

```bash
# 构建 Docker 镜像
docker build -t cool-admin-vue .

# 运行容器
docker run -d -p 80:80 cool-admin-vue
```

### Nginx 部署

1. 构建生产版本：`pnpm build`
2. 将 `dist` 目录内容部署到 Nginx 服务器
3. 配置 Nginx（参考项目根目录的 `nginx.conf`）

### 静态部署

使用 `pnpm build-static` 构建静态版本，可部署到任意静态资源服务器。
