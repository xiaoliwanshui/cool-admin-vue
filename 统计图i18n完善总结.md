## 🔧 技术修复

- 修复了 `hot-search.vue` 组件中的 TypeScript 类型错误

- 在 `hot-search.vue` 中使用 `parseInt()` 将字符串类型的 count 转换为数字
- 月份标签使用 i18n 插值语法：`t('{i}月', { i: i + 1 })`

## 🌍 国际化架构遵循

### 使用规范
✅ **组件中使用 `useI18n()` hook 获取 `t` 函数**
```typescript
import { useI18n } from 'vue-i18n';
const { t } = useI18n();
```

✅ **模板中使用 `{{ t('key') }}` 或 `{{ $t('key') }}`**
```vue
<span>{{ t('热搜排行榜') }}</span>
<span>{{ $t('浏览量') }}</span>
```

✅ **翻译键使用简短、描述性的中文命名**
- 避免冗长的键名
- 使用直观的中文描述
- 保持键名的一致性

✅ **新增功能时所有 UI 文本都使用 i18n 函数**
- 无硬编码中文文本
- 支持动态语言切换
- 保持多语言同步
